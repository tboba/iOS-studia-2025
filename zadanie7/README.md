# zadanie 7

Aby skorzystać z autoryzacji klienta, potrzebne jest uzupelenienie pliku Info.plist o:

- ID klienta GitHuba (wzięty z https://github.com/settings/developers)
- Sekret klienta GitHuba (wzięty z https://github.com/settings/developers).
- ID Klienta Google (wpisany w GIDClientID oraz CFBundleURLSchemes wewnatrz klucza google-oauth)

Aby móc zalogować się z serwera, wymagane dane podstawowego pierwszego uzytkownika to:
login: admin
haslo: admin123

Istnieje rowniez opcja rejestracji.
